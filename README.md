# IronHabit

Versione incrementale del progetto sviluppata con Expo SDK 54 e TypeScript.

## Schermate iniziali

- Home
- Allenamento
- Profilo

È presente una prima base grafica condivisa in tema scuro, con icone e componenti
riutilizzabili per intestazioni e contenuti introduttivi.

## Contenuti demo

Le schermate hanno ora una prima base di contenuti:

- riepilogo settimanale nella Home;
- prime schede allenamento nella sezione Allenamento;
- statistiche e obiettivo attuale nel Profilo.

## Avvio

```bash
npm install
npm run start
```
