# IronHabit

IronHabit e una mobile app per organizzare esercizi, schede, sessioni pianificate e obiettivi personali.
Permette di avviare allenamenti guidati, completare serie, usare timer di recupero e salvare i workout svolti.
La Home mostra saluto, statistiche essenziali, grafici e agenda settimanale delle sessioni.
Lo storico permette di consultare i workout completati e monitorare progressi, volume e dettagli delle serie.
Tutti i dati restano salvati in locale sul dispositivo, senza backend o servizi esterni.

## Avvio

```bash
npm ci
npm start
```
